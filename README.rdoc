==Wagn: Wiki on Wheels

Wagn is an open web platform for collaborating to build open web systems.

Using simple building blocks called cards, Wagneers can construct web systems ranging from simple content managers to custom project trackers to fully integrated web suites.  

To learn more about installation, see http://wagn.org/Installation

To learn more about configuration, see http://wagn.org/Configuration

For other documentation, see http://wagn.org/Documentation

For community support, see http://wagn.org/Support
